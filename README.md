# 🚗 Describe Car Amount - AI Destekli Araç Analizi

Bu proje, Google'ın **Gemini 1.5 Flash** modelini kullanarak yüklenen görsellerdeki araçları analiz eden ve kullanıcıya detaylı bilgi sunan bir web uygulamasıdır. **Cursor** kullanılarak geliştirilmiş ve **Vercel** üzerinde yayına alınmıştır.

## ✨ Özellikler

* **Yapay Zeka Analizi:** Fotoğraftaki araç sayısını ve türlerini tespit eder.
* **Hızlı Yanıt:** Gemini 1.5 Flash modelinin hızı sayesinde anlık sonuçlar.
* **Modern Arayüz:** Kullanıcı dostu ve şık tasarım.
* **Responsive:** Hem mobil hem de masaüstü cihazlarla tam uyumlu.

## 🚀 Teknolojiler

* **Frontend:** HTML5, CSS3, JavaScript
* **AI Model:** [Google Gemini 1.5 Flash](https://ai.google.dev/)
* **Deployment:** [Vercel](https://vercel.com/)
* **Version Control:** Git & GitHub
